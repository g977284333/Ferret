# 趋势功能用户体验改进

## 问题1：进入页面时显示已采集的关键词

**现象**：进入趋势页面时，热门和已采集关键词就显示了fitness

**说明**：这是**正常行为**
- 之前采集过的关键词数据保存在数据库中
- 页面加载时会自动从数据库读取并显示
- 这样可以：
  - 快速查看历史采集的关键词
  - 继续分析之前采集的数据
  - 无需重复采集

**如果需要清空**：
- 可以手动删除数据库中的趋势数据
- 或者添加"清空数据"功能（可选）

## 问题2：采集完成后自动加载图表

**需求**：采集完成后，图表应该自动选择刚采集的关键词并加载

**实现**：
- ✅ 采集完成后，自动选择第一个采集的关键词
- ✅ 自动加载该关键词的趋势图表
- ✅ 显示提示信息："已自动加载关键词 'xxx' 的趋势图表"

**用户体验**：
- 采集完成后立即看到结果
- 无需手动选择关键词
- 提升操作效率
