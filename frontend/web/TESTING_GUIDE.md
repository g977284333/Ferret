# 测试指南

## 快速测试

### 1. 基础测试

```bash
# 测试所有页面
python test_pages.py

# 测试所有API
python test_api.py

# 完整工作流程测试
python test_full_workflow.py
```

### 2. 手动测试

#### 测试首页
1. 访问 http://localhost:5000
2. 检查页面是否正常加载
3. 查看统计信息是否正确显示
4. 测试快速开始表单

#### 测试数据采集
1. 访问 http://localhost:5000/scrape
2. 添加关键词（如：productivity）
3. 点击"开始采集"
4. 观察进度条是否更新
5. 检查结果统计

#### 测试机会列表
1. 访问 http://localhost:5000/opportunities
2. 测试搜索功能
3. 测试筛选功能
4. 测试排序功能
5. 测试分页功能
6. 测试导出功能

#### 测试机会详情
1. 在列表页面点击"查看详情"
2. 检查信息是否完整
3. 检查评分详情是否正确
4. 测试导出功能

#### 测试配置页面
1. 访问 http://localhost:5000/config
2. 调整评分权重
3. 修改筛选阈值
4. 保存配置
5. 验证配置是否生效
