# 趋势功能测试清单

## ✅ 服务启动成功

服务已成功启动，运行在：
- http://localhost:5000
- http://127.0.0.1:5000

## 🧪 测试步骤

### 1. 访问趋势页面

在浏览器中打开：`http://localhost:5000/trends`

应该看到：
- ✅ 页面正常加载
- ✅ 导航栏有"搜索趋势"链接
- ✅ 页面显示"搜索趋势分析"标题

### 2. 测试关键词管理

- [ ] 在关键词输入框中输入：`fitness`
- [ ] 按Enter键或点击"添加"按钮
- [ ] 验证关键词显示在关键词列表中
- [ ] 点击关键词旁边的×按钮，验证可以删除

### 3. 测试平台选择

- [ ] Google Trends选项应该已选中
- [ ] 其他平台显示为"待实现"（这是正常的）

### 4. 测试时间范围选择

- [ ] 点击时间范围下拉菜单
- [ ] 选择不同的时间范围（如"过去3个月"）
- [ ] 验证选择正常

### 5. 测试启动采集

- [ ] 确保至少有一个关键词（如`fitness`）
- [ ] 点击"开始采集"按钮
- [ ] 验证进度卡片显示
- [ ] 验证状态指示器显示"运行中"
- [ ] 观察进度更新

**注意**：Google Trends采集可能需要一些时间，并且需要网络连接到Google。

### 6. 测试数据展示

采集完成后：
- [ ] 查看"热门关键词"列表是否更新
- [ ] 查看"已采集关键词"列表是否更新
- [ ] 验证关键词出现在下拉菜单中

### 7. 测试分析功能

- [ ] 在"已采集关键词"列表中点击某个关键词的"分析"按钮
- [ ] 验证分析结果是否显示
