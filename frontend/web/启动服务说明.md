## ⚠️ 常见问题

### 问题1：窗口立即关闭

**原因**：代码有错误导致启动失败

**解决方法**：
1. 在PowerShell中手动运行 `python app.py`
2. 查看完整的错误信息
3. 把错误信息发给我

### 问题2：提示端口被占用

**错误信息**：
```
OSError: [WinError 10048] 通常每个套接字地址(协议/网络地址/端口)只允许使用一次
```

**解决方法**：
1. 找到占用端口的程序：
   ```powershell
   netstat -ano | findstr :5000
   ```
2. 关闭那个程序，或者修改端口（编辑 `app.py` 最后一行改为 `port=5001`）

### 问题3：提示模块未找到

**错误信息**：
```
ModuleNotFoundError: No module named 'xxx'
```

**解决方法**：
```powershell
pip install flask flask-cors pytrends pandas
```

## 🔍 验证服务是否启动

### 方法1：查看终端输出

如果看到：
```
* Running on http://0.0.0.0:5000
```
说明服务已启动。

### 方法2：检查端口

在**新的**PowerShell窗口中运行：
```powershell
netstat -ano | findstr :5000
```
如果看到输出，说明服务正在运行。

### 方法3：访问首页

在浏览器打开：`http://localhost:5000`
- ✅ 如果看到首页，说明服务正常
- ❌ 如果显示"拒绝连接"，说明服务未启动

2. **如果服务启动后立即退出**
