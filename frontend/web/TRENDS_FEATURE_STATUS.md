## 📝 使用说明

### 采集趋势数据
1. 输入关键词（支持多个，用逗号分隔）
2. 选择平台（默认Google Trends）
3. 选择时间范围（默认12个月）
4. 点击"开始采集"
5. 等待采集完成

### 查看已采集关键词
- 采集完成后，关键词会自动出现在"已采集关键词"列表中
- 可以点击"刷新已采集关键词"手动刷新

### 分析关键词
- 在"已采集关键词"列表中，点击关键词旁边的"分析"按钮
- 查看增长率、趋势分数、统计信息等

### 查看热门关键词
- 热门关键词会自动显示在"热门关键词"列表中
- 按增长率从高到低排序
- 可以点击"刷新热门关键词"手动刷新
