# 功能验证计划

## 验证目标

验证当前系统的核心功能是否正常工作，并测试不同场景下的表现。

## 验证项目

### 1. 数据采集功能验证
- [ ] 测试不同关键词
- [ ] 验证数据完整性
- [ ] 验证去重功能
- [ ] 验证数据保存

### 2. 数据分析功能验证
- [ ] 验证评分模型计算
- [ ] 验证筛选阈值
- [ ] 验证排序功能
- [ ] 验证结果输出

### 3. 参数调整测试
- [ ] 测试不同评分权重
- [ ] 测试不同筛选阈值
- [ ] 验证配置文件的生效

### 4. 结果合理性验证
- [ ] 分析Top机会是否合理
- [ ] 验证评分是否准确
- [ ] 检查数据质量

## 验证步骤

### 步骤1：基础功能测试
使用默认参数运行系统，验证基本功能。

### 步骤2：不同关键词测试
测试不同领域的关键词，看系统表现。

### 步骤3：参数调整测试
调整评分权重和阈值，看结果变化。

### 步骤4：结果分析
分析输出结果，评估系统准确性。
