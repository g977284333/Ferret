# 趋势数据说明

## 📊 value字段的含义

**value字段**代表**Google Trends的搜索热度值**，这是一个**相对值**（0-100），不是绝对搜索量。

### 具体说明：

1. **相对值，不是绝对值**
   - value = 100：表示该日期是该关键词在选定时间范围内的**最高搜索热度**
   - value = 50：表示搜索热度是最高值的50%
   - value = 0：表示搜索量不足，无法显示（或数据缺失）

2. **时间范围相关**
   - 如果选择"过去12个月"，那么100代表这12个月内的最高值
   - 如果选择"过去7天"，那么100代表这7天内的最高值
   - **不同时间范围的数据不能直接对比**

3. **地区相关**
   - 默认是全球数据
   - 不同地区的搜索热度值也是相对的

### 如何理解value值：

- **value > 80**：搜索热度很高，可能是热门趋势
- **value 50-80**：搜索热度中等
- **value < 50**：搜索热度较低
- **value = 0**：数据缺失或搜索量不足

## ⚠️ 为什么会有重复数据？

可能的原因：

1. **多次采集同一关键词**
   - 如果多次采集同一个关键词，数据库中会有多条记录
   - 虽然使用了`INSERT OR REPLACE`，但如果日期格式不同，可能不会覆盖

2. **日期格式不一致**
   - 例如："2024-01-01" 和 "2024-01-01 00:00:00" 会被视为不同的记录

3. **没有去重逻辑**
   - 导出时没有对数据进行去重处理

## 💡 如何让导出的数据更有价值？

### 建议1：添加分析字段
导出时应该包含：
- **增长率**：最近30天 vs 前30天的增长率
- **趋势**：上升/下降/稳定
- **平均热度**：时间范围内的平均搜索热度
- **波动性**：搜索热度的波动程度

### 建议2：数据清洗
- **去重**：按关键词+日期去重，保留最新的数据
- **排序**：按日期排序
- **过滤**：移除value=0的数据（可选）

### 建议3：导出格式优化
- **按关键词分组**：每个关键词一个sheet
- **添加统计信息**：每个关键词的统计摘要
- **添加图表**：在Excel中自动生成趋势图表
