## 快速概览

### 技术栈（推荐⭐）
- **后端**：Flask
- **前端**：Tailwind CSS + Flowbite
- **图表**：Chart.js
- **表格**：DataTables

### 备选方案
- **前端**：Bootstrap 5 + jQuery（最快开发）

### 核心功能
1. 数据采集控制
2. 机会列表查看
3. 机会详情展示
4. 配置管理

### 开发时间
- **MVP**：7-12天（1-2周）
