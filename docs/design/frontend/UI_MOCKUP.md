# 前端UI设计稿

## 设计理念

- **简洁明了**：信息层次清晰
- **操作便捷**：常用功能易访问
- **视觉美观**：现代化设计风格
- **响应迅速**：实时反馈用户操作

## 响应式设计

### 桌面端（> 1200px）
- 完整布局
- 多列显示
- 侧边栏导航

### 平板端（768px - 1200px）
- 单列布局
- 顶部导航
- 适配触摸

### 移动端（< 768px）
- 单列布局
- 底部导航
- 简化功能

## 交互细节

### 1. 采集进度
- 实时进度条
- 百分比显示
- 预计剩余时间
- 可暂停/停止

### 2. 机会列表
- 表格排序（点击表头）
- 实时搜索
- 筛选器展开/收起
- 分页导航

### 3. 评分可视化
- 进度条显示各维度分数
- 颜色编码（绿色=高，黄色=中，红色=低）
- 权重显示

### 4. 操作反馈
- 按钮点击动画
- 加载状态提示
- 成功/失败提示
- 确认对话框

## 技术实现细节

### 前端框架
- Bootstrap 5（UI组件）
- jQuery（DOM操作）
- Chart.js（图表）
- DataTables（表格）

### 后端API
- Flask RESTful API
- WebSocket（实时进度，可选）
- JSON数据格式

### 数据交互
- AJAX异步请求
- 实时数据更新
- 错误处理
- 加载状态管理
