# Markdown 编辑和预览工具推荐

## 🎯 推荐工具（按推荐程度排序）

### 1. Obsidian（强烈推荐）⭐

**优点**：
- ✅ **免费**（个人使用）
- ✅ **实时预览**（所见即所得）
- ✅ **强大的链接系统**（适合课程学习）
- ✅ **插件生态丰富**
- ✅ **本地存储**（数据安全）
- ✅ **支持中文**（完美支持）
- ✅ **双向链接**（可以建立知识网络）
- ✅ **标签系统**（方便组织文档）

**缺点**：
- ❌ 需要学习一些概念（链接、标签等）
- ❌ 界面可能稍显复杂

**适用场景**：
- 学习课程文档
- 做作业和笔记
- 建立知识体系

**下载地址**：https://obsidian.md/

**使用建议**：
1. 将 `docs/courses/entrepreneur-skills/` 文件夹作为 Obsidian 的库
2. 可以建立文档之间的链接
3. 使用标签管理作业进度
4. 使用插件增强功能（如：任务管理、日历等）

---

### 2. Typora（付费，但体验最好）

**优点**：
- ✅ **最佳预览体验**（真正的所见即所得）
- ✅ **界面简洁美观**
- ✅ **支持中文**（完美支持）
- ✅ **导出功能强大**（PDF、HTML等）
- ✅ **表格编辑方便**
- ✅ **数学公式支持好**

**缺点**：
- ❌ **收费**（$14.99，一次性购买）
- ❌ 没有链接系统（不如 Obsidian）

**适用场景**：
- 需要高质量预览
- 需要导出PDF
- 写作和编辑

**下载地址**：https://typora.io/

---

### 3. VS Code + Markdown 插件（如果你用VS Code）

**优点**：
- ✅ **免费**
- ✅ **如果你已经在用VS Code**，无需切换
- ✅ **插件丰富**
- ✅ **支持中文**

**推荐插件**：
- **Markdown Preview Enhanced**：强大的预览功能
- **Markdown All in One**：增强编辑功能
- **Markdown PDF**：导出PDF

**缺点**：
- ❌ 预览需要分屏（不如 Obsidian 和 Typora 方便）
- ❌ 需要配置插件

**适用场景**：
- 如果你已经在用 VS Code 开发
- 想要在同一个工具中编辑代码和文档

**安装方法**：
1. 打开 VS Code
2. 搜索并安装 "Markdown Preview Enhanced"
3. 打开 Markdown 文件，按 `Ctrl+Shift+V` 预览

## 🎯 针对课程学习的建议

### 使用 Obsidian 的最佳实践

1. **建立知识网络**
   - 在文档中使用 `[[]]` 创建内部链接
   - 例如：`[[01-ENTREPRENEUR_SKILLS_DEVELOPMENT]]`
   - 可以快速跳转到相关文档

2. **作业管理**
   - 使用标签：`#作业 #第1周 #已完成`
   - 使用任务：`- [ ] 作业1.1`
   - 使用看板：管理作业状态

3. **笔记和反思**
   - 为每个作业创建笔记
   - 记录学习过程
   - 建立反思文档

4. **进度跟踪**
   - 创建进度跟踪文档
   - 使用任务列表
   - 使用日历视图

## 📝 快速开始指南

### 使用 Obsidian（推荐）

1. **下载安装**
   ```
   访问：https://obsidian.md/
   下载并安装
   ```

2. **创建库**
   ```
   打开 Obsidian
   选择 "Open folder as vault"
   选择：D:\workspace\GC\Ferret\docs\courses\entrepreneur-skills
   ```

3. **开始使用**
   ```
   左侧会显示所有文档
   点击文档即可查看和编辑
   使用 Ctrl+E 切换编辑/预览模式
   ```

4. **推荐设置**
   ```
   设置 → 编辑器 → 显示行号
   设置 → 编辑器 → 自动保存
   设置 → 外观 → 选择主题
   ```
