# 课程作业系统

> **目标**：通过明确的作业，确保学以致用，验证学习效果，有过程、有结果

## 🎯 作业系统说明

### 为什么需要作业？

1. **执行力不足**：明确的作业任务，帮助你克服拖延
2. **边界不清晰**：明确的输出要求，让你知道做到什么程度
3. **学以致用**：每个作业都是实际应用，不是纸上谈兵
4. **验证效果**：通过作业结果，验证你是否真正掌握
5. **过程记录**：记录学习过程，看到自己的成长

### 作业系统特点

- ✅ **明确的任务**：每个作业都有清晰的任务描述
- ✅ **明确的输出**：每个作业都有明确的输出要求
- ✅ **明确的检查标准**：每个作业都有检查清单
- ✅ **明确的提交格式**：统一的作业提交模板
- ✅ **进度跟踪**：记录完成情况，看到进度

## 📋 作业清单

### 阶段1：基础建立（第1-4周）

#### 第1周：商业思维基础

**作业1.1：商业模式分析**
- **任务**：选择一个成功产品，分析其商业模式
- **输出**：商业模式分析报告（1-2页）
- **检查标准**：
  - [ ] 明确回答了"它解决了什么问题？"
  - [ ] 明确回答了"它如何创造价值？"
  - [ ] 明确回答了"它如何传递价值？"
  - [ ] 明确回答了"它如何获取价值？"
  - [ ] 有数据支撑（市场规模、用户数等）
- **提交格式**：见下方"作业提交模板"
- **完成时间**：本周内

**作业1.2：市场定义**
- **任务**：选择一个市场，定义市场边界
- **输出**：市场定义文档（1页）
- **检查标准**：
  - [ ] 列出了5个感兴趣的市场
  - [ ] 选择了1个市场
  - [ ] 明确了市场边界（包括什么、不包括什么）
  - [ ] 划分了细分市场（至少3个）
- **提交格式**：见下方"作业提交模板"
- **完成时间**：本周内

**作业1.3：市场规模分析**
- **任务**：分析选定市场的规模
- **输出**：市场规模分析报告（1-2页）
- **检查标准**：
  - [ ] 估算了TAM（总市场规模）
  - [ ] 估算了SAM（可服务市场规模）
  - [ ] 估算了SOM（可获得市场规模）
  - [ ] 分析了市场增长（历史、未来、驱动因素）
  - [ ] 有数据来源（至少3个）
- **提交格式**：见下方"作业提交模板"
- **完成时间**：本周内

## 💡 作业完成建议

### 1. 时间管理

- **每天固定时间**：每天固定1-2小时做作业
- **小步快跑**：不要等到最后一天，每天做一点
- **优先级管理**：先做重要的作业

### 2. 质量保证

- **不要追求完美**：先完成，再优化
- **有数据支撑**：不要空想，要有数据
- **有实际应用**：不是纸上谈兵，要实际应用

- **总结收获**：总结学到了什么
