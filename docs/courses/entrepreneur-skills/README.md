# 个人创业者核心能力提升课程

> **课程目标**：系统性地提升个人创业者的核心能力，从0到1完成创业

## 📚 课程结构

本课程包含以下核心文档，按学习顺序排列：

### 1. 核心能力体系（必读）
**文件**：`01-ENTREPRENEUR_SKILLS_DEVELOPMENT.md`

**内容**：
- 7层能力体系全景图
- 21个核心能力详解
- 12周能力提升路径
- 综合实践项目

**学习时间**：2-3小时阅读 + 12周实践

**适合人群**：所有个人创业者

---

### 2. 市场分析能力（核心能力之一）
**文件**：`02-MARKET_ANALYSIS_SKILLS_DEVELOPMENT.md`

**内容**：
- 市场分析的本质和方法
- 需求挖掘的具体方法
- 需求分析的实践框架
- 常见陷阱和解决方案

**学习时间**：1-2小时阅读 + 4周实践

**适合人群**：需要提升市场分析能力的创业者

---

### 3. 实践练习手册（必做）
**文件**：`03-PRACTICAL_EXERCISES.md`

**内容**：
- 12周完整实践计划
- 21个具体练习
- 练习模板和工具
- 学习路径建议

**学习时间**：12周持续实践

**适合人群**：所有个人创业者

---

### 3.5 作业系统（必做）⭐
**文件**：`ASSIGNMENTS.md`

**内容**：
- 完整的作业清单（14个作业）
- 明确的作业任务和输出要求
- 作业提交模板
- 作业检查标准
- 进度跟踪表

**学习时间**：12周持续完成作业

**适合人群**：所有个人创业者（特别是执行力不足、边界不清晰的学习者）

**为什么重要**：
- ✅ 明确的作业任务，帮助你克服拖延
- ✅ 明确的输出要求，让你知道做到什么程度
- ✅ 明确的检查标准，验证你是否真正掌握
- ✅ 进度跟踪，看到自己的成长

---

### 4. 思维突破（重要）
**文件**：`04-BEYOND_DATA_ANALYSIS.md`

**内容**：
- 跳出数据分析的思维局限
- 从"数据驱动"到"假设驱动"
- 系统的新定位
- 新的工作流程

**学习时间**：30分钟阅读

**适合人群**：陷入技术路径依赖的开发者

---

### 5. 行动指南（重要）
**文件**：`05-STOP_DEVELOPING_START_USING.md`

**内容**：
- 停止开发，开始使用
- 系统的新定位
- 立即行动计划
- 常见陷阱和解决方案

**学习时间**：30分钟阅读

**适合人群**：一直在开发系统，没有实际使用的开发者

---

## 🎯 学习路径

### 路径1：完整学习（推荐）

**适合**：想要系统提升所有能力的创业者

**步骤**：
1. **第1周**：阅读核心能力体系（01）
   - 理解7层能力体系
   - 评估自己当前能力
   - 制定学习计划

2. **第2周**：阅读市场分析能力（02）
   - 理解市场分析本质
   - 学习需求挖掘方法
   - 开始实践练习

3. **第3-14周**：完成实践练习（03）
   - 按12周计划完成所有练习
   - 记录学习过程
   - 持续反思和优化

4. **持续**：阅读思维突破（04）和行动指南（05）
   - 避免常见陷阱
   - 保持正确方向

**总时间**：14周（3个月）

---

### 路径2：快速入门

**适合**：时间有限，想快速上手的创业者

**步骤**：
1. **第1天**：阅读核心能力体系（01）的前3部分
   - 理解能力体系
   - 选择重点提升的能力

2. **第2-3天**：阅读市场分析能力（02）
   - 学习核心方法
   - 开始第一个练习

3. **第4天开始**：完成实践练习（03）的前4周
   - 完成基础练习
   - 建立基础能力

4. **持续**：根据实际需要选择深入学习

**总时间**：4周（1个月）

---

### 路径3：问题导向

**适合**：有具体问题需要解决的创业者

**步骤**：
1. **识别问题**：明确你的具体问题
   - 市场分析能力不足？
   - 需求挖掘能力不足？
   - 陷入技术路径依赖？

2. **选择文档**：根据问题选择对应文档
   - 市场分析问题 → 02
   - 思维局限问题 → 04
   - 行动问题 → 05

3. **深入学习**：深入学习相关部分
   - 理解核心概念
   - 完成相关练习

4. **扩展学习**：根据需要扩展学习
   - 学习相关能力
   - 完成综合项目

**总时间**：根据问题灵活调整

---

## 📋 学习建议

### 1. 学习顺序

**必读顺序**：
1. 核心能力体系（01）- 建立整体认知
2. 实践练习手册（03）- 开始实践
3. 市场分析能力（02）- 深入学习核心能力
4. 思维突破（04）- 避免陷阱
5. 行动指南（05）- 立即行动

### 2. 学习方法

**理论 + 实践**：
- 先理解理论（阅读文档）
- 再实践应用（完成练习）
- 持续反思优化（记录和总结）

**小步快跑**：
- 不要追求完美
- 快速开始，快速迭代
- 从简单到复杂

**持续学习**：
- 每天学习一点
- 每周完成一个练习
- 每月完成一个阶段

### 3. 学习工具

**记录工具**：
- 笔记软件（Notion、Obsidian等）
- 文档工具（Markdown、Word等）

**实践工具**：
- 系统工具（Ferret系统）
- 市场分析工具（Google Trends等）
- 需求挖掘工具（Reddit、Twitter等）

**学习资源**：
- 推荐书籍（见各文档）
- 推荐网站（见各文档）
- 推荐工具（见各文档）
